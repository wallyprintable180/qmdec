## 🔍 Troubleshooting Common Issues

If you experience problems, check these items first:

### Program does not open
Ensure your computer runs a 64-bit version of Windows. Try downloading the file again if the download interrupted.

### Files show as locked
Close the music player currently using the files. qmdec cannot modify files that other programs keep open.

### Missing tags
Some encrypted files contain no metadata. The program performs basic tagging as a default. You might need to add album art manually in your music player.

### Error messages
Restart the application if an error stalls the process. Clear the list and try a smaller batch of files to identify problematic tracks.
